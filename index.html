Plumber Bookings Web Application

This application allows plumbers to register and become part of a network. This will allow users to see which plumbers are in the network and available.

Users will then be able to book plumbers in 1 of 3 slots per day. The user will need to enter their name and contact details as well as a day they would like to book the plumber for and a slot in that day.

Once a plumber has been booked the plumber will be able to see who has booked them for what day and which slot in that day the user needs that plumbers service.

Shoe Catalogue API and Shoe Catalogue Web Application

This API allows a user to enter data through an application called POSTMAN. The user of this API can:

  • view a list all shoes that are currently in the database
  • view all shoes with a given brand
  • view all shoes with a given size
  • view all shoes with a given brand and size
  • update the stock levels when a shoe is sold
  • and add a new shoe to the inventory
This API can be used in conjunction with a front-end application that will provide the user with an interface to do all the above mentioned.

Waiter Availability Web Application

This application allows a waiter to enter their name directly into the url and they will be added to the database of existing waiters. Waiters will then be able to select which days they are available to work and this will be saved with their names.
On the admin side, the admin can view which waiter is working on which day as well as whether more waiters are needed for a particular day.
Waiter information is stored on MongoDB.
